# Contributing

Thanks for wanting to contribute to @alpha/devtools-detector! This guide explains how to get started quickly.

## 1) Setup

```bash
git clone <your-fork-url>
cd devtools-detector
npm install
```

## 2) Develop

- Make code changes in `src/`.
- Build locally:

```bash
npm run build
```

## 3) Run CI locally (optional)

Use `npm run build` to verify the TypeScript build.

## 4) Create commit

- Use clear commit messages.
- Bump version in `package.json` when releasing.

## 5) Pull request

1. Push branch to your fork.
2. Open PR to `main` branch.
3. Ensure GitHub CI passes.

## Code style

- Keep code simple and readable.
- Prefer robust error-checking.
- Keep comments up to date with behavior.

## Questions

If you’re unsure about behavior or strategy, open an issue first.
